Role: Designers

Problems & Recommendations

1. Contrast between text and background color

The first problem is about the contrast between text and its background color. As we can see from the image below. The title “University of Michigan Research” is light blue. However, the background color is dark blue. So it is a little bit hard to see it. So if we can change the text color to white, it will become easier.

2. Missing alt text

Ten images in this web page have no alt text. Every image should have an alt text about what this image is.

3. Not so consistent layout

As we can see from the two images below: for the first image, the fonts of the title and paragraph are the same. However, for the second image, the fonts of two parts are different. So I recommend that the headings should have the same typeface in one page.
